Projects

Research & applied work.

Sentinel-2 Python NDWI Remote Sensing

Wetland Permanence Classification

Automated classification of wetland hydroperiods (permanent, vegetated, seasonal) using temporal variability of NDWI derived from multi-seasonal Sentinel-2 imagery across South Dakota prairie potholes.

Key Methods

  • Coefficient of Variation analysis on NDWI time-series
  • Multi-seasonal Sentinel-2 image processing
  • Python automation for large-scale raster workflows
  • Hydroperiod classification (permanent / seasonal / vegetated)

ArcGIS Pro Arcade OpenStreetMap Spatial Analysis

Chicago Park Accessibility Analysis

Created a unified park feature layer for Chicago by integrating official municipal data with OpenStreetMap to evaluate urban green space equity and identify underserved neighborhoods for park development priorities.

Key Methods

  • CRS reprojection: WGS 1984 → NAD 1983 Illinois East
  • Arcade expressions to resolve 12-digit Census ID mismatches
  • CLOSEST spatial join to resolve geometric nulls
  • Topology validation (Must Not Overlap rules)
